Aspire .NET Learning Repository

Purpose
This repository is provided for learning and experimentation with .NET projects and distributed e-shop patterns. Use it to explore code structure, build processes, and basic deployment patterns.

Contents
- AspireSample/ — small example projects and exercises.
- eshop-distributed/ — a sample distributed e-shop application demonstrating microservices, messaging, and basic integration patterns.

Getting started
1. Install .NET SDK (recommended 6.0 or later) from https://dotnet.microsoft.com.
2. Open a project folder (for example, AspireSample) in Visual Studio or VS Code.
3. From the project folder, build and run:
   dotnet build
   dotnet run

Notes
- This repo is for study and experimentation only. Feel free to modify code and add notes.
- Commit personal changes to branches; avoid committing secrets or local config files.

Contributing
Create branches for changes, open pull requests with clear descriptions, and keep commits focused on single ideas.

Happy learning!
